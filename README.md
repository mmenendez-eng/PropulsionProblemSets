# Sutton Problem 3-22

This script analyzes the thrust coefficient as a function of nozzle expansion ratio for k = 1.3 and p1/p3 = 50.

Key results:
- Identifies optimum expansion ratio
- Shows pressure term sign change at p2 = p3
- Visualizes momentum vs pressure contributions

Tools:
- NumPy
- SciPy
- Matplotlib
